# Qwen3-TTS Portable - Русскоязычная версия

Портативная версия Qwen3-TTS для Windows с полностью русскоязычным интерфейсом и поддержкой стриминга.

## Возможности

- **Пресеты голосов** - 9 предустановленных голосов с возможностью настройки стиля
- **Клонирование голоса** - создание голоса из референсного аудио
- **Дизайн голоса** - генерация голоса по текстовому описанию
- **Потоковая генерация** - отображение прогресса в реальном времени
- **Полная изоляция** - все файлы хранятся в папке приложения
- **Поддержка 10+ языков** - включая русский, английский, китайский и др.

## Системные требования

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| GPU VRAM | 4GB (0.6B) | 8GB+ (1.7B) |
| RAM | 8GB | 16GB+ |
| Место на диске | 10GB | 20GB |
| ОС | Windows 10 | Windows 11 |

## Установка

### 1. Скачайте и распакуйте

Распакуйте архив в любую папку (избегайте путей с кириллицей).

### 2. Запустите установку

```
install.bat
```

Установщик автоматически:
- Загрузит Python 3.12.8 Embeddable
- Установит PyTorch с поддержкой вашей видеокарты
- Установит все необходимые зависимости
- Настроит Flash Attention 2 (для RTX 30xx/40xx/50xx)

### 3. Запустите приложение

```
run.bat
```

Приложение будет доступно по адресу: http://127.0.0.1:7860

## Структура папок

```
portable/
├── install.bat          # Установщик
├── run.bat              # Запуск приложения
├── app.py               # Основное приложение

├── python/              # Встроенный Python (создается при установке)

└── temp/                # Временные файлы
```

## Использование

### Пресеты голосов (CustomVoice)

1. Выберите вкладку "Пресеты голосов"
2. Введите текст для синтеза
3. Выберите язык и голос
4. (Опционально) Добавьте стиль: "Говорить радостно"
5. Нажмите "Сгенерировать"

### Клонирование голоса (Base)

1. Выберите вкладку "Клонирование голоса"
2. Загрузите референсное аудио (3-10 секунд)
3. Введите текст референсного аудио
4. Введите текст для синтеза
5. Нажмите "Клонировать и озвучить"

### Дизайн голоса (VoiceDesign)

1. Выберите вкладку "Дизайн голоса"
2. Введите текст для синтеза
3. Опишите желаемый голос на английском или русском
4. Нажмите "Сгенерировать"

## Решение проблем

### Ошибка "CUDA out of memory"

- Используйте модель 0.6B вместо 1.7B
- Закройте другие программы, использующие GPU
- Уменьшите параметр "Макс. токенов"

### Медленная генерация

- Установите Flash Attention 2 (RTX 30xx+)
- Используйте меньшую модель (0.6B)

### Модели не загружаются

- Проверьте интернет-соединение
- Убедитесь, что в папке `models` достаточно места
